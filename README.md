## 简介
### 初始化项目
clone项目

    npm install
### 运行
    npm serve
### 编译
    npm build
访问：http://localhost:8080/
### 制作自己的简历
找到resume.json文件，按照自己的情况修改相应的信息即可。
### 打印预览
打开上述链接，在浏览器中按
    ctrl+p
## 其他
蓝色主题版：自行修改 ./scss/style.scss 中的变量。

注意：这里用了很多css3，并不支持低版本ie。
